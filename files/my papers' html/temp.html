Federated Learning with Patient-Annotated Data in Epileptic Seizure Detection

Amin Aminifar, Jonathan Dan, David Atienza
International Joint Conference on Neural Networks (IJCNN), 2025

Download PDF

Visual abstract

Abstract

We present a federated learning framework that leverages patient-annotated EEG data to enable personalized epileptic seizure detection while preserving data privacy. The method coordinates model updates across distributed clients and integrates annotation quality to improve robustness and performance.
